# Serving-only dependencies for the Cloud Run image.

# Kept separate from requirements.txt because image size drives cold-start

# the exception. Excluded here: xlrd (ingestion only), google-cloud-bigquery and
# db-dtypes are kept because the prediction log writes to BigQuery.
#
# PyTorch is installed from the CPU-only wheel index. The default wheel bundles
# CUDA libraries totalling several GB, which are dead weight on Cloud Run where
# there is no GPU. See docker/Dockerfile for the --index-url that selects it.
